Extra Information

 This server is running v3.0 of GNUMP3d the open source MP3/Ogg/Movie streaming server - distributed under the terms of the GNU General Public License.

 This server was primarily written by Steve Kemp, and improved thanks to suggestions from users and contributions of code, theme designs, and logos - Many more details can be found upon the project homepage at http://www.gnump3d.org/.

 If you require any assistence in running the software, have any customization requests, or ideas for features then please consider the following links as starting points:

  • Frequently Asked Questions - and answers.
  • Mailing Lists are a good source of help.
  • The Author answered as free time permits.

 If you believe you have discovered a bug please take the time to report them so that all users might benefit. Similarly if you have discovered what you believe is a security issue please report it responsibly; full credit would be given upon the website for any such issues raised.

 Feedback is always appreciated via mail to one of the mailing lists, or the author directly. If you are a Freshmeat user you may rate the software there.
